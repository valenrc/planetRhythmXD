# Planet Rhythm - Changelog

## Versión 0.0.2m

### Nuevas Features

#### Menú Principal Rediseñado
- rework completo del menú principal con nuevo diseño visual
- integración del logo de Planet Rhythm en el menú
- nuevo ícono de aplicación

#### Música en Pantalla de Resultados
- música dedicada para la pantalla de resultados (endscreen)

#### Nuevos Assets Visuales
- nuevo sprite de alien en estilo pixel art
- nuevos assets de personaje alien

### Mejoras
- reestructuración general del proyecto para mejor organización del código
- limpieza de assets y fuentes no utilizadas
- actualización de la documentación del proyecto

### Correcciones
- fix de crash del conductor causado por BPM en la escena de nivel
- fix parcial de la pantalla de resultados (endscreen)
- corrección de paths de scripts globales y AudioStream para mejor portabilidad

---

## Versión 0.0.2

### Nuevas Features

#### Sistema de Calibración de Audio
- menú de calibración de offset para ajustar la latencia del sistema.
- offset global configurable para sincronizar el audio con las notas.
- mejora en la precisión del timing para la versión web.

#### Mejoras Visuales
- nueva animación en la pantalla de resultados (endscreen)
- animación mejorada del planeta durante el gameplay para mejor feedback visual

### Optimizaciones
- mejor rendimiento general del juego
- optimización de texturas y assets

### Correcciones
- fixes generales de estabilidad
- correcciones en el sistema de timing
- mejoras en la respuesta del input

---

## Versión 0.0.1 - Release Inicial

- 3 canciones jugables: Flow, Spaced Out, Space Hardstyle
- Sistema de puntuación con combo y accuracy
- Selector de skins de planetas
- Velocidad de notas y planeta configurable
- Sistema de HP (health points)
- Soporte para dos tipos de notas
- Interfaz con selector de niveles
